BOB ROSS PAINTINGS ANALYSIS
---------------------------

Lecture for discussing reading files, generating plots, etc, by analyzing
information about the various paintings created on "The Joy of Painting".

This lecture makes use of raw data from FiveThirtyEight[1], in particular the
bob_ross dataset.  The data were modified to be presented in a different format
for this lecture.  The original data were made available under a CC-BY 4.0
license[2].

[1] https://github.com/fivethirtyeight/data
[2] https://creativecommons.org/licenses/by/4.0/
